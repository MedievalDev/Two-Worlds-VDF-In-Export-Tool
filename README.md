# Two-Worlds-VDF-In-Export-Tool
Modding tools for Two Worlds 1 (2007). Convert 3D models between TW1's proprietary VDF format and standard OBJ. Export game models for editing in Blender/3ds Max/Maya, or import custom models into the game. Includes full reverse-engineered VDF/NTF format documentation. Python 3.8+, no dependencies.
